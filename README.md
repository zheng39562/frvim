# 安装和使用

## 使用
	* 遵循原则：
		* 尽量使用原插件快捷键。
		* 在不破坏原插件的基础上会有一些对快捷键扩展(本质是调用原有快捷键or指令)
		* 多插件冲突，会保留某一个插件的快捷键(保留暂无原则，以使用方便为准)
	* 具体使用建议查看vimrc:主要考虑到readme更新处于一种滞后的状态. 而vimrc一定是最新的原则。所以不再这里去介绍具体的使用和细节.
